 Static Web Project


Dit project is een eenvoudige webshop gemaakt voor het vak Static Web.  
De webshop toont sport- en streetwear producten in een overzichtelijke layout.

Het doel van dit project is om te werken met:
- HTML
- CSS
- Een duidelijke structuur
- Een responsieve en gebruiksvriendelijke pagina




Het project bestaat uit de volgende bestanden en mappen:

- index.html → hoofdpagina (home)
- producten.html → overzicht van de producten
- css/style.css → styling van de website
- images/ → afbeeldingen van de producten en hero image




1. Eerst werd de HTML-structuur aangemaakt (nav, hero, producten, footer).
2. Daarna werd CSS toegevoegd voor layout, kleuren en typografie.
3. Er werd gebruik gemaakt van Flexbox en Grid voor de productweergave.
4. Afbeeldingen werden toegevoegd via de images-map.
5. Navigatie werkt met ankers (Home, Producten, Contact).
6. Het project werd geüpload naar GitHub.




- Kleurenschema: zwart, wit en grijs voor een moderne uitstraling.
- Font: Montserrat via Google Fonts.
- Producten worden weergegeven in een grid (3 per rij).
- Footer bevat contactinformatie.
