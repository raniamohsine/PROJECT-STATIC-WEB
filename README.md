 Dit project is een eenvoudige webshop gemaakt voor het vak Static Web.
De website is opgebouwd met HTML en CSS en maakt gebruik van Flexbox en Grid voor de layout.

De structuur bestaat uit een homepagina, een productpagina, een CSS-bestand en een map met afbeeldingen.
De producten worden overzichtelijk weergegeven en de website is responsief.
